# 🌤️ SkyCast Pro: Reactive Weather Dashboard

A professional, object-oriented weather application built with Python and Streamlit. This app features real-time weather data fetching, browser-based geolocation, and a reactive UI that changes its theme based on current weather conditions.

## 🚀 Features

*   **Dual Search Modes:** Search by city name or use the "📍 Use GPS" button to fetch weather for your exact location.
*   **Reactive Backgrounds:** The app's CSS dynamically updates with smooth transitions (e.g., stormy purple for thunderstorms, bright blue for clear skies).
*   **Object-Oriented Design:** Built using a modular class structure for easy maintenance and scalability.
*   **Secure API Handling:** Fully integrated with Streamlit Secrets to prevent API key leaks.
*   **Responsive Metrics:** View temperature, humidity, and weather conditions with live icons from OpenWeatherMap.

## 🛠️ Tech Stack

*   **Language:** Python 3.x
*   **Framework:** [Streamlit](https://streamlit.io/)
*   **API:** [OpenWeatherMap API](https://openweathermap.org/api)
*   **Libraries:** `requests`, `streamlit-js-eval`
